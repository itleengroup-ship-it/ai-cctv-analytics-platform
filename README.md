# AI CCTV Analytics Platform

## Enterprise AI Surveillance SaaS Platform

### Overview
AI-powered surveillance SaaS platform designed to transform traditional CCTV systems into intelligent cloud-managed security ecosystems.

---

## Architecture

Camera → DVR/NVR → Local Gateway PC → Tailscale VPN → Cloud AI Server → SaaS Dashboard

---

## Features

- AI Video Analytics
- YOLO Detection
- Human & Vehicle Detection
- Intrusion Detection
- Self-Learning AI
- Secure VPN Streaming
- Cloud Dashboard
- Multi-Site Monitoring
- Real-Time Alerts

---

## Technologies

- Python
- YOLO
- OpenCV
- FastAPI
- Docker
- RTSP
- Tailscale VPN
- GPU AI Inference

---

## System Workflow

1. CCTV Cameras capture video
2. DVR/NVR records streams
3. Local Gateway PC manages streams
4. Tailscale secures communication
5. Cloud AI analyzes video
6. Dashboard receives alerts

---

## Vision

Building a scalable enterprise AI surveillance ecosystem powered by cloud AI and self-learning analytics.
